JSON 格式化工具

JSON（JavaScript Object Notation）是Web开发中最常用的数据交换格式。API返回的JSON通常是压缩在一行的，难以阅读。使用此工具可以一键展开为缩进清晰、易于阅读的格式。

常见场景

  • API调试：格式化服务器返回的JSON响应
  • 配置文件：美化package.json等配置
  • 数据导出：JSON数组转CSV用Excel打开
  • 语法检查：快速定位逗号缺失、引号不匹配
